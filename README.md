# SwiftKotlination
Experimental comparison app, implementing the same Architecture and Design Patterns in iOS Swift and Android Kotlin.

## Frameworks

### iOS (Swift)

- SnapKit
- RxSwift
- Alamofire
- RxAlamofire

### Android (Kotlin)

- Anko
- Dagger
- RxKotlin
- RxAndroid
- Retrofit
- Moshi

## Architecture

- MVVM

## Design Patterns

- Coordinator
- Dependency Injection
